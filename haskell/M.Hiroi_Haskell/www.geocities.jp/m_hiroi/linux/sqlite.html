はじめに

SQLite は D. Richard Hipp 氏が開発しているパブリックドメインな軽量のリレーショナルデータベース管理システム (RDBMS) です。一般的な RDBMS (たとえば MySQL など) はサーバとして動作させるのですが、SQLite はアプリケーションに組み込んで使用することができます。中小規模なシステムであれば、SQLite でも十分なパフォーマンスが得られるようです。

また、SQLite はコンパクトでインストールも簡単なので、初心者でもさくさくと RDBMS の学習を進めていくことができます。このページでは SQLite を使って、RDBMS の基本を勉強していきたいと思っております。RDBMS に関する M.Hiroi の覚え書にすぎませんが、興味のある方はお付き合いくださいませ。

●インストールと起動

Ubuntu 系の OS であれば、次のコマンドで SQLite をインストールすることができます。 

sudo apt-get install sqlite3

端末で sqlite3 ファイル名 を入力すると、端末から指定したファイルをデータベースとして操作することができます。終了コマンドは .exit または .quit (.q でも OK) です。 

mhiroi@mhiroi-VirtualBox:~/sqlite$ sqlite3 test.sql
SQLite version 3.11.0 2016-02-15 17:29:24
Enter ".help" for usage hints.
sqlite> .exit
