●●● ソートアルゴリズムデモ ●●●

シェーカーソートコームソート単純挿入ソートシェルソート

Java による各種ソートアルゴリズムのデモンストレーションです。 画面をクリックするとソートを開始します。比較しているデータは、 下に赤線と青線を引いて表しています。シェーカーソートの場合は、 青線でソート範囲を表し、赤線で比較しているデータのひとつを表しています。 ソート完了後、画面をクリックすると新しいデモが始まります。

●ソースファイルのダウンロード

  • シェーカーソート
  • コームソート
  • 単純挿入ソート
  • シェルソート
